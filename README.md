## HTTP codes

### 1xx: Informational

### 2xx: Sucess
    - 200: SUCESS
    - 201: CREATED

### 3xx: Redirection
    - 301: Moved Permanently
    - 302: Moved

### 4xx: Client Error
    - 400: Bad Request
    - 401: Unauthorized
    - 404: Not Found

### 5xx: Server Error
    - 500: Internal Server Error
